# scala-cli-sbt-mongo-crud-update

## Description
Creates a user with credentials
and lists collection `test`.

## Tech stack
- scala
- sbt
  - mongo connector
- mongo

## Docker stack
- mongodb:latest
- openjdk:8-jdk-alpine

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
